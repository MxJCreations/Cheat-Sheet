HTML Table TAGS

HTML TAG

Description
Table

Creates a table. This has both opening and closing tags.
thead

This is the head of the table. Add a 'tr' and then a 'th' with 'colspan' to cover how many columns the heading goes across. H2 is a good size heading for this.
tr

'tr' stands for table row. It has opening and closing tags.
th

'th' stands for table heading. It has opening and closing tags.
td

'td' stands for table data. It has opening and closing tags.
tbody

'tbody' stands for table body. The main contents of the table are entered here, between the opening and closing tags.
